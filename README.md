# GBFRVFL: Granular-Ball Computing-Based Fuzzy Random Vector Functional Link Network



Please cite the following paper if you are using this code. Reference: A. Quadir, A. Rahaman, P. N. Suganthan and M. Tanveer. "GBFRVFL: Granular-Ball Computing-Based Fuzzy Random Vector Functional Link Network." 2026 IEEE World Congress on Computational Intelligence.


%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% 
The experiments are executed on a computing system possessing Python 3.11 software, an Intel(R) Xeon(R) CPU E5-2697 v4 processor operating at 2.30 GHz with 128-GB Random Access Memory (RAM), and a Windows-11 operating platform.

We have put a demo of the “GBFRVFL” model with the "bank" dataset

In this demonstration, the codes are run for fixed hyperparameter for the "bank" dataset
c_1 = 0.1
N = 23
act = 6

For detailed hyperparameter setting, please refer to the paper.

Description of files: 
main.py: This is the main file to run selected algorithms on datasets. In the path variable specificy the path to the folder containing the codes and datasets on which you wish to run the algorithm. 
gen_ball.py: Generation of Granular ball and calculating the membership
RVFL.py: Solving the optimization problem.

For the detailed experimental setup, please follow the paper. If you find any bugs/issues, please write to A. Quadir (mscphd2207141002@iiti.ac.in).
